Listas em Python

O que é uma Lista?

Uma Lista em Python é uma coleção ordenada e mutável de itens. É uma das estruturas de dados mais versáteis e frequentemente utilizadas em Python.

Características principais:

  • Ordenada (mantém a ordem de inserção)
  • Mutável (pode adicionar, remover ou modificar itens)
  • Permite itens duplicados
  • Pode conter diferentes tipos de dados

# Criando uma Lista
frutas = ["maçã", "banana", "laranja", "uva"]

# Acessando elementos
print(frutas[0])  # Output: maçã
print(frutas[-1])  # Output: uva

# Modificando elementos
frutas[1] = "morango"

# Adicionando elementos
frutas.append("pêra")

# Removendo elementos
frutas.remove("laranja")

# Fatiando a lista
print(frutas[1:3])  # Output: ['morango', 'uva']

# Comprimento da lista
print(len(frutas))  # Output: 4

# Iterando sobre a lista
for fruta in frutas:
    print(fruta)

# Verificando se um item está na lista
if "maçã" in frutas:
    print("Maçã está na lista!")

Tuplas em Python

O que é uma Tupla?

Uma Tupla em Python é uma coleção ordenada e imutável de itens. É semelhante a uma lista, mas não pode ser modificada após a criação.

Características principais:

  • Ordenada (mantém a ordem de inserção)
  • Imutável (não pode ser modificada após a criação)
  • Permite itens duplicados
  • Pode conter diferentes tipos de dados

# Criando uma Tupla
coordenadas = (10, 20)

# Tupla com diferentes tipos de dados
pessoa = ("João", 30, "Engenheiro")

# Acessando elementos
print(coordenadas[0])  # Output: 10
print(pessoa[1])  # Output: 30

# Tentando modificar uma tupla (isso causará um erro)
# coordenadas[0] = 15  # TypeError: 'tuple' object does not support item assignment

# Desempacotando uma tupla
nome, idade, profissao = pessoa
print(f"Nome: {nome}, Idade: {idade}, Profissão: {profissao}")

# Comprimento da tupla
print(len(coordenadas))  # Output: 2

# Iterando sobre a tupla
for item in pessoa:
    print(item)

# Verificando se um item está na tupla
if "João" in pessoa:
    print("João está na tupla!")

# Contando ocorrências de um item
numeros = (1, 2, 2, 3, 4, 2)
print(numeros.count(2))  # Output: 3

# Encontrando o índice de um item
print(numeros.index(3))  # Output: 3

Set() em Python

O que é um Set?

Um Set em Python é uma coleção não ordenada de itens únicos. É útil para eliminar itens duplicados de uma sequência e realizar operações matemáticas como união e interseção.

Características principais:

  • Itens únicos (não permite duplicatas)
  • Não ordenado
  • Mutável (pode adicionar ou remover itens)
  • Pode conter diferentes tipos de dados imutáveis

# Criando um Set
frutas = {"maçã", "banana", "laranja", "maçã"}
print(frutas)  # Output: {'maçã', 'banana', 'laranja'}

# Adicionando um item
frutas.add("uva")

# Removendo um item
frutas.remove("banana")

# Operações de conjunto
set1 = {1, 2, 3, 4, 5}
set2 = {4, 5, 6, 7, 8}

# União
print(set1 | set2)  # Output: {1, 2, 3, 4, 5, 6, 7, 8}

# Interseção
print(set1 & set2)  # Output: {4, 5}

# Diferença
print(set1 - set2)  # Output: {1, 2, 3}

# Diferença simétrica
print(set1 ^ set2)  # Output: {1, 2, 3, 6, 7, 8}

# Verificando se um item está no set
if "maçã" in frutas:
    print("Maçã está no set!")

# Iterando sobre o set
for fruta in frutas:
    print(fruta)

# Comprimento do set
print(len(frutas))

Dicionários em Python

O que é um Dicionário?

Um Dicionário em Python é uma coleção não ordenada de pares chave-valor. É útil para armazenar dados associativos e realizar pesquisas rápidas por chave.

Características principais:

  • Pares chave-valor
  • Chaves únicas
  • Mutável (pode adicionar, modificar ou remover pares)
  • Chaves podem ser de diferentes tipos imutáveis

# Criando um Dicionário
pessoa = {
    "nome": "Ana",
    "idade": 30,
    "profissão": "Engenheira"
}

# Acessando um valor
print(pessoa["nome"])  # Output: Ana

# Adicionando um novo par chave-valor
pessoa["cidade"] = "São Paulo"

# Modificando um valor
pessoa["idade"] = 31

# Removendo um par chave-valor
del pessoa["profissão"]

# Verificando se uma chave existe
if "nome" in pessoa:
    print("A chave 'nome' existe no dicionário!")

# Obtendo todas as chaves
print(pessoa.keys())

# Obtendo todos os valores
print(pessoa.values())

# Obtendo todos os pares chave-valor
print(pessoa.items())

# Iterando sobre o dicionário
for chave, valor in pessoa.items():
    print(f"{chave}: {valor}")

# Obtendo um valor com um valor padrão se a chave não existir
print(pessoa.get("altura", "Não especificada"))

# Atualizando o dicionário com outro dicionário
pessoa.update({"profissão": "Desenvolvedora", "idiomas": ["Português", "Inglês"]})

# Comprimento do dicionário
print(len(pessoa))
